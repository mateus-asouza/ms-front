# COMO RODAR O PROJETO

1. Instalar todas as dependencias indicada pelo package.json
```
### npm install
```
2. Rodar o projeto
```
### npm run dev

```
## SEQUENCIA PARA CRIAR O PROJETO

1. Criar o arquivo package
```
### npm init
```

2. Criar o projeto com Next.js
```
### npm install next react react-dom
```
3. Rodar o projeto
```
### npm run dev
```
